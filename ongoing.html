Portable Diabetic Retinopathy Detection System With Advanced AI Inference Techniques

Tonmoy Paul et al. — Final Year Design Project

Developed an AI-enabled portable edge device using Raspberry Pi 5, Pi camera, and 20D double aspheric lens for real-time color fundus photography (CFP) capture. Engineered an ensemble deep learning architecture (Xception + ResNet50 Gated Hybrid) integrated via a custom PyQt6 GUI, achieving 98.93% accuracy for ICDR diabetic retinopathy classification. Managed clinical data collection combining APTOS 2019 with an IRB-approved, locally sourced dataset from hospitals in Bangladesh.
